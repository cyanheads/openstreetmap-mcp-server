# 0.1.1 — 2026-05-23

First functional release. Implements the full Nominatim + Overpass tool surface on top of `@cyanheads/mcp-ts-core`.

## Added

- **`nominatim_geocode`** — forward geocoding via Nominatim: free-form query or structured address fields, country filtering, layer/featureType filtering, extra OSM tags, preferred language override, results ordered by importance score
- **`nominatim_reverse`** — reverse geocoding: lat/lon to address with zoom-level control (building → country), layer filtering, extra tags, language preference
- **`nominatim_lookup`** — batch address resolution for up to 50 known OSM IDs (N/W/R prefix format); reports `not_found` list for unmatched IDs
- **`overpass_query_nearby`** — POI/feature search within a radius using `amenity` shortcut or `tag_key`+`tag_value`, up to 50 km, results include OSM type/ID and full tag set
- **`overpass_query_bbox`** — same interface as nearby but bounded by a rectangular bounding box
- **`overpass_query_raw`** — arbitrary Overpass QL execution; server injects `[timeout:N]` when absent
- Nominatim service layer (`src/services/nominatim`) — HTTP client, search, reverse, and lookup methods
- Overpass service layer (`src/services/overpass`) — query builder, executor, element normalizer
- Structured error contracts: `no_results`, `no_coverage`, `invalid_id_format`, `invalid_tag`, `query_timeout`, `rate_limited`, `query_error`, `result_too_large` — all with actionable recovery hints
- OSM attribution string on every response (`Data © OpenStreetMap contributors, ODbL 1.0`)
- Private instance support via `NOMINATIM_BASE_URL`, `OVERPASS_BASE_URL` env vars
- MCPB bundle (`manifest.json`) and one-click install badges for Claude Desktop, Cursor, VS Code
- `server.json` registry metadata with stdio and HTTP package entries
