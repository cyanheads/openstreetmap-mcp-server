# 0.1.3 — 2026-05-23

## Changed

- **`package.json`**: added `mcpName`, `bugs`, `homepage`, `author`, `funding`, `packageManager` fields; expanded `keywords`; normalized `repository.url` to `git+https://...git` form.
- **`server.json`**: `name` updated to `io.github.cyanheads/nominatim-mcp-server` (registry-qualified form).
- **`manifest.json`**: version bumped from stale `0.1.1` → `0.1.2` (was not bumped in the v0.1.2 release).
- **`Dockerfile`**: OCI labels filled in — `title`, `description`, and `source` were empty or missing.
- **`bunfig.toml`**: added with standard Bun install/run defaults.

## Fixed

- **`overpass_query_raw`**: added preflight check for `[out:json]` directive — without it Overpass returns XML, `JSON.parse` throws, and the error surfaces as an opaque `InternalError`. Now fails fast with a descriptive `query_error` before the network call.
